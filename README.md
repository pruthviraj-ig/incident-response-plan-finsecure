# Cyber Incident Response Plan – FinSecure Ltd

This project presents a structured Incident Response Plan developed for a fictional FinTech organisation (FinSecure Ltd).

##  Scenario
Phishing Attack Leading to Credential Compromise

##  What This Includes
- Incident Response Lifecycle (Detection → Containment → Eradication → Recovery)
- Incident Analysis and Root Cause Identification
- CIA Impact Assessment
- Incident Timeline
- Communication Plan (GDPR aligned)
- Risk Mapping to Risk Assessment
- Control Improvements & Recommendations
- Lessons Learned
- Appendices (Incident Log, Evidence Register, Response Checklist)

## Framework Alignment
- ISO/IEC 27001
- NIST Incident Response Lifecycle
- GDPR
